# Isochron
A PCB badge that has a piezo buzzer and an OLED, making it usable as a metronome! It also has two capacitive touch buttons, with one upping the BPM and the other generating a random BPM. "Isochron" is derived from "_isochronos"_, meaning "equal time" in Greek.
